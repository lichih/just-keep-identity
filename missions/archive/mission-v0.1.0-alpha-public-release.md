# Mission: v0.1.0-alpha 公開發布與社群推廣 (Public Release & Promotion)

## 🎯 目標 (Goals)
將 JKI 從本地開發環境推向全球開源社群，獲取首批 20+ 則有效回饋，驗證 UX 語義並建立專案知名度。

---

## 📝 任務清單 (Tasks)

### Phase 1: 發布素材準備 (Assets Preparation)
- [ ] **視覺化 Demo**：編寫 [vhs](https://github.com/charmbracelet/vhs) 腳本並產出高品質的 Demo GIF (展示搜尋高亮、Copy 提示、Handshake)。
- [ ] **多語系 README**：建立 `README.md` (英文) 與 `README.zh-TW.md` (繁體中文)，包含快速安裝與核心理念。
- [ ] **GitHub Release**：針對 `v0.1.0-alpha` Tag 撰寫 Release Notes，提供 macOS 二進位檔下載。

### Phase 2: 社群推廣 (Community Outreach)
- [ ] **Hacker News (Show HN)**：撰寫以 "Local-first MFA for CLI" 為核心的投遞文案。
- [ ] **Reddit (r/rust, r/commandline)**：分享 JKI 的設計細節（如：Rust + age + IPC）。
- [ ] **國內社群 (PTT/Dcard/iT邦幫忙)**：以「提升開發者生產力」為題進行分享。
- [ ] **Twitter/X Showcase**：配合 #rustlang 標籤發布 Demo GIF。

### Phase 3: 反饋收集與對帳 (Feedback & Audit)
- [ ] **Issue 追蹤**：建立 Issue Template (Bug Report / Feature Request)。
- [ ] **量化指標核對**：確認是否達成 `ROADMAP.md` 中 v0.1.x 要求的「20 則操作建議」。

---

## 📈 成功定義 (Definition of Done)
1.  GitHub Repo 擁有完整的 README 與 Demo 視覺。
2.  在至少 3 個主要開源社群完成首發。
3.  累積獲得 > 20 則來自非開發者的有效操作建議或 Bug 回饋。
4.  達成 v0.1.x 的「macOS 穩定運行」量化指標。
